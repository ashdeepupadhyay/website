Discord Chat Bot

Technology Used : Nodejs,Mongodb,google custom search API

Github
Created a chat bot for a server with some functionality.
  • If a user sends 'hi', the bot will reply 'hey'
  • if a user sends '!google search_keyword', and it'll reply with the top five links for the search_keyword for example !google cricket it will return with top five links from google custom search api for cricket.
  • if a user sends '!recent search', and it'll reply with a list of similar search in the user's history for example you have given following commands !google apple fruit,!google football,!google banana fruit and then you did ! recent fruit the result will be banana,apple.
